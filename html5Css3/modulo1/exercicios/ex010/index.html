Trabalhando com Listas

Listas Ordenadas

Para criar as listas ordenadas, usamos a Tag <ol>

E para indicar os itens desse tipo de lista, usamos a Tag <li>

  1. item 1
  2. item 2
  3. item 3

Listas não ordenadas

Para criar as listas não ordenadas, usamos a Tag <ul>

E para indicar os itens desse tipo de lista, usamos a Tag <li>

  • item 1
  • item 2
  • item 3
